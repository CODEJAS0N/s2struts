S2Struts

  • Enables the interaction of S2 with existing applications using Struts. (Basic Usage)
  • Prevents the sprawling struts-config.xml problem. (Unconfigured S2Struts Reference)
  • Allows the usage of easy to test POJO as an Action. (Using POJO as an Action)
  • Helps with non-POJO Action testing as well. (S2StrutsUnit Reference)

S2Struts not only creates a bridge between existing applications and S2, but provides many functions to ease overall web application development.

A Quickstart tutorial is prepared to give you a guided walk through of web application development using S2Struts.

Let's give it a try.
